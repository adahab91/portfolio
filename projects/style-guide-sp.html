Style Guide

Given task:

"Please design a style guide in your chosen graphics software. The style guide should consist of a few pages."

This is more advanced than a simple style guide or style tile as it is specific to the website design, including interactivity and placement of elements in the actual website pages. This is given to other members in a team when others are working on the same website design and/or given to the Web Developer.

In this school project I chose to create a style guide for an Interior Design website.
